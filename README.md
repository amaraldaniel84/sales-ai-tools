# Sales AI Tools

A collection of utilities for simulating and testing generative AI capacity and performance on Google Cloud.

## Applications

- **Gemini Internal Cache Tester (`GeminiImplicitCacheTester.html`)**: A diagnostic tool designed to test and verify Gemini's implicit caching behavior and performance.
- **Vertex AI GSU Simulator (`googlecloud_simulador_de_gsus.html`)**: A capacity planning tool that calculates the required Generative AI Scale Units (GSUs) based on traffic distribution and token volume assumptions.
- **SLA & Overflow Visualizer (`simulador_visual_de_sla_pt.html`)**: An interactive animation showing how Provisioned Throughput affects service level agreements (SLAs) and handles traffic overflow.

## Usage

These are standalone HTML applications. To use them, simply open the respective `.html` file in any modern web browser.
